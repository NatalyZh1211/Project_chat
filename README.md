# Проект ЧАТ
Чат (консольное приложение)

**Разработал:** Журавлева Наталья

**Проверил:** Глебов Евгений

**Функционал приложения**:
1. регистрация пользователей - логин, пароль, имя;
1. вход в чат по логину/паролю;
1. смена пароля;
1. удаление аккаунта; 
1. отправка сообщений конкретному пользователю;
1. отправка сообщений всем пользователям;
1. получение сообщений от конкретного пользователя;
1. получение сообщений от всех пользователей.

 
**Описание выбранной идеи решения** 
Реализованы четыре класса 
- класс одиночного пользователя;
- класс с массивом пользователейпользователей;
- класс одиночного сообщения;
- класс с массивом сообщений.
При регистрации пользователя записывается информация в массив пользователей, при удалении аккаунта - удаляется.
При авторизации изменяется статус пользователя в массиве.
При смене пароля изменяется пароль пользователя с определенным именем в массиве пользователей.
Каждое сообщение в массиве сообщений содержит информацию о номере отправителя, получателя и строку сообщения.
При обмене сообщениями в массиве сообщений производится поиск сообщений по номеру получателя и отправителя и выводятся сообщения в консоль.
